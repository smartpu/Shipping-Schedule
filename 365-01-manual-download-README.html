船期网页手动下载工具使用说明

工具简介

船期网页手动下载工具 是一个用于从 hangyun365.com 网站手动下载船期网页的辅助工具。由于网站有反爬措施，禁止使用自动化工具（如 Selenium），因此需要手动操作。本工具提供了进度跟踪、状态管理等功能，让手动下载过程更加高效。

⚠️ 重要提示：由于网站有反爬措施，禁止使用自动化工具。请使用本手动下载工具。

工具版本

本工具提供两个版本：

基础版 (365-01-manual-download.html)

  • 包含 URL 数量：46 个
  • 适用场景：常用航线，快速下载
  • 预计时间：约 12-15 分钟

完整版 (365-01-manual-download-full.html)

  • 包含 URL 数量：168 个（完整航线列表）
  • 适用场景：需要下载所有航线数据
  • 预计时间：约 45-60 分钟

快速开始

第一步：选择工具版本

根据需求选择使用基础版或完整版：

  • 基础版：双击打开 365-01-manual-download.html
  • 完整版：双击打开 365-01-manual-download-full.html

第二步：准备保存文件夹

确保在项目根目录下存在 365-view-source 文件夹。如果不存在，请先创建该文件夹。

第三步：开始下载

按照页面上的说明，逐个打开网页并保存。详细操作步骤见下方。

详细操作步骤

步骤 1：打开网页

  1. 在工具页面找到要下载的 URL
  2. 点击"打开网页"按钮
  3. 会在新标签页中打开对应的网页
  4. 等待页面完全加载（约 10 秒）

步骤 2：保存网页

  1. 在浏览器中按 Ctrl+S（Windows）或 Cmd+S（Mac）保存网页
  2. 保存位置：选择 365-view-source 文件夹
  3. 文件名格式：pol_code-pod_code.html
    • 从 URL 参数中提取：pol_code 和 pod_code
    • 示例：
      • URL: https://www.hangyun365.com/Shipping?pol_code=CNTAO&pod_code=HKHKG
      • 文件名: CNTAO-HKHKG.html
  4. 如果文件已存在，选择覆盖即可

步骤 3：更新进度

  1. 保存完成后，回到工具页面
  2. 点击"标记为已保存"按钮更新进度
  3. 工具会自动记录你的进度（保存在浏览器本地存储）
  4. 进度条和统计信息会自动更新

步骤 4：继续下一个

  1. 建议每保存一个网页后，等待 10 秒再处理下一个，避免请求过快
  2. 重复步骤 1-3，直到所有网页下载完成

工具功能

核心功能

  • ✅ 进度跟踪：自动记录已保存的网页，刷新页面后进度不会丢失
  • ✅ 可视化进度条：实时显示下载进度百分比
  • ✅ 统计信息：显示总计、已保存、待处理数量
  • ✅ 状态标记：每个 URL 都有清晰的状态标识（已保存/待处理）
  • ✅ 一键打开：点击按钮即可在新标签页打开网页
  • ✅ 重置功能：点击"已保存"数字可清除所有进度，方便重新开始

预计时间

基础版（46 个网页）

  • 每个网页：约 15-20 秒（打开 10 秒 + 保存 5-10 秒）
  • 总共：约 12-15 分钟

完整版（168 个网页）

  • 每个网页：约 15-20 秒（打开 10 秒 + 保存 5-10 秒）
  • 总共：约 45-60 分钟

常见问题

Q1: 打开网页后显示反爬提示

A: 这是正常的，网站检测到自动化工具。使用手动工具（本 HTML 文件）可以正常访问。如果遇到反爬提示，请：

  1. 关闭提示窗口
  2. 等待页面完全加载（约 10 秒）
  3. 正常保存网页即可

Q2: 文件名格式不对

A: 文件名应该从 URL 参数中提取。例如：

  • URL: https://www.hangyun365.com/Shipping?pol_code=CNTAO&pod_code=HKHKG
  • 文件名: CNTAO-HKHKG.html

如果浏览器自动填充的文件名不对，请手动修改为正确的格式。

Q3: 进度丢失

A: 进度保存在浏览器本地存储中。如果清除浏览器数据，进度会丢失，但可以重新标记。建议：

  1. 不要清除浏览器数据
  2. 如果进度丢失，可以重新标记已保存的文件
  3. 或者点击"已保存"数字清除所有进度，重新开始
