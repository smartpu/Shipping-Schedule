# Market Watch · 市场观察 使用说明

## 工具简介

**Market Watch · 市场观察** 是一款集成的海运市场分析工具，通过一份 Excel 文件即可联动运价趋势、航线供给与 AI 决策，轻松完成跨板块洞察。工具整合了运费走势可视化、航线运力分析和 AI 智能分析三大核心模块，为海运定价与航线决策提供数据支持。

---

## 快速开始

### 前置准备

1. **Excel 数据文件**：准备包含 `market` 和 `schedule` 两个工作表的 Excel 文件（`.xlsx` 或 `.xls` 格式）
2. **市场周报（可选）**：准备 PDF 格式的市场周报（如 Alphaliner Newsletter、Linerlytica Weekly 等）
3. **AI API 密钥（可选）**：如需使用 AI 分析功能，需准备 DeepSeek / KIMI (Moonshot) / 通义千问 (Qwen) API 密钥

### 基本使用流程

1. 双击打开 `365-04-market-watch.html`
2. 在顶部上传区域：
   - 点击"选择Excel文件"，选择包含 `market` 和 `schedule` 工作表的 Excel 文件
   - 点击"上传市场周报 (PDF)"，可一次选择多份 PDF 报告（可选）
3. 等待数据加载完成后，三个模块的筛选控件将自动显示
4. 根据需求使用各模块功能进行分析
5. 如需导出报告，点击"导出整页 PDF"按钮

## 核心功能模块

### Module 01 · 运费走势图 (RATE TRENDS CHART)

**功能定位**：多港口报价、区间提示与时段筛选，一屏掌握市场温度

#### 主要功能

- **多选筛选**：
  - 支持多选区域和港口
  - 系统自动匹配 20GP / 40GP 报价
  - 自动显示每个选中港口的价格区间（格式：港口名：20GP 最小值 ~ 最大值，40GP 最小值 ~ 最大值）

- **时间范围筛选**：
  - 可设定开始时间和结束时间
  - 查看特定周期的价格走势
  - 时间控件与图表标题同行显示

- **可视化展示**：
  - 交互式折线图，清晰展示价格趋势
  - 同时显示 20GP（实线）和 40GP（虚线）价格走势
  - 支持多港口对比，不同港口使用不同颜色区分
  - 图表信息框显示选中港口数量、时间节点数和价格区间

#### 数据要求

- Excel 文件需包含 `market` 工作表
- 工作表需包含以下列：
  - **日期列**：日期格式（A列或包含"日期"、"date"、"时间"等关键词）
  - **目的港列**：港口名称（D列或包含"港口"、"port"、"目的港"等关键词）
  - **区域列**：区域分类（I列或包含"区域"、"region"、"地区"等关键词）
  - **20GP 列**：20GP 价格（列名包含 "20GP"、"20gp"、"GP20"）
  - **40GP 列**：40GP 价格（列名包含 "40GP"、"40gp"、"GP40"）

### Module 02 · 航线运力分析 (SAILING SCHEDULE PULSE)

**功能定位**：四级筛选联动运力、派船与趋势，精准锁定上周+本周+未来三周供给

#### 主要功能

- **四级筛选体系**：
  - **区域** → **子区域** → **国家/地区** → **港口**
  - 支持多选，筛选条件逐级联动
  - 筛选后自动更新表格和图表

- **航线搜索**：
  - 在表头"共舱船公司"列输入关键词
  - 支持中文输入，实时过滤
  - 与四级筛选条件组合使用

- **三种视图模式**：
  - **全部**：同时显示运力和派船数
  - **运力**：仅显示运力（TEU）
  - **派船**：仅显示派船数量

- **周别统计**：
  - 固定显示上周+本周+未来三周（共5周）
  - 周别定义：一周从周日到周六（如：2025/11/16 周日 ~ 2025/11/22 周六 为 2025/47）
  - 表头显示周别（YYYY/WW，周数格式为两位数如 2026/01）和具体日期范围
  - 支持查看历史船期信息

- **表格结构**：
  - **航线备注列**：显示航线备注信息（对应 schedule 表 K 列）
  - **共舱船公司列**：显示共舱船公司信息，悬停可查看完整路径（对应 schedule 表 Y 列）
  - **周别列**：显示上周+本周+未来三周（共5周）的运力和派船数据

- **趋势图表**：
  - 选择区域后自动显示运力与派船趋势图
  - 堆积柱形图展示各航线运力分布
  - 折线图展示派船总数变化
  - 图表标签格式：`航线备注 - 共舱船公司`
  - 双 Y 轴设计，便于对比分析

- **详细信息查看**：
  - 悬停表格周别单元格可查看详细船期信息（船名航次、运力、开船日期）
  - 悬停共舱船公司列可查看完整路径信息
  - 筛选后自动显示统计汇总行

#### 数据要求

- Excel 文件需包含 `schedule` 工作表
- 工作表需包含以下列：
  - **目的港列**：港口名称（第2列或包含"港口"、"目的港"、"pod"等关键词）
  - **区域列**：区域分类（包含"区域"、"大区"、"region"等关键词）
  - **子区域列**：子区域分类（包含"子区域"、"次区域"、"sub"等关键词）
  - **国家/地区列**：国家或地区名称（包含"国家地区"、"国家/地区"、"国家"、"country"等关键词）
  - **航线列**：共舱船公司或航线名称（F列或包含"共舱"、"船公司"、"航线"、"route"等关键词）
  - **航线ID列**：航线标识（F列或包含"航线id"、"航线ID"、"route id"、"group_id"等关键词）
  - **航线备注列**：航线备注信息（K列，对应表格中的"航线备注"列）
  - **完整路径列**：完整路径信息（Y列，对应表格中"共舱船公司"列的悬停提示）
  - **周别列**：周别代码（N列或包含"周别"、"周次"、"周数"、"week"等关键词，格式如：202547 或 2025年第47周）
  - **船名航次列**：船名和航次（H列或包含"船名"、"航次"、"vessel"等关键词）
  - **开船日期列**：开船日期（I列或包含"开船"、"船期"、"离港"、"etd"、"时间"等关键词）
  - **运力列**：运力（TEU）（O列或包含"运力"、"teu"、"舱位"等关键词）

### Module 03 · AI 趋势分析 (AI TREND ANALYSIS)

**功能定位**：DeepSeek × KIMI × 通义千问 三模型交叉验证航线策略、供需与价格建议

#### 主要功能

- **三模型支持**：
  - **DeepSeek**：支持 deepseek-chat、deepseek-v3、deepseek-reasoner、deepseek-r1、deepseek-coder 等模型
  - **KIMI (Moonshot)**：支持 moonshot-v1-8k、moonshot-v1-32k、moonshot-v1-128k、moonshot-v1-k2 等模型
  - **通义千问 (Qwen)**：支持 qwen-turbo、qwen-plus、qwen-max、qwen-max-longcontext 等模型
  - 可切换对比不同模型的分析结论
  - 配置自动保存到浏览器本地存储

- **其他影响因素输入**：
  - **收货情况**：填写当周及未来2周订舱情况、旺季/淡季判断、重点流向表现、装载率预估等
  - **码头情况**：填写起运港/目的港的拥堵情况、靠泊等待平均时间、天气或其他码头运营影响
  - **额外运力**：填写具体船公司在某周临时加开/取消/减舱的情况（航线运力分析未覆盖的部分）
  - **市场运费**：填写现行报价，市场上较为激进的高价或低价（如 XX 公司报价表）
  - **其他事件**：填写调高关税、出口退税、查验力度、环保政策变化等特殊事件

- **外部数据抓取**：
  - **燃油行情（Ship & Bunker · 新加坡）**：
    - VLSFO 0.5%、MGO 0.1%、IFO380
    - 显示价格和 WoW（周环比）变化
    - 点击"更新燃油价格"按钮手动刷新
  - **中国出口集装箱运价指数（CCFI）**：
    - 综合指数及12条主要航线指数
    - 显示本期、上期和 WoW 变化
    - 点击"更新 CCFI"按钮手动刷新
  - **WCI 主要航线现货价（Drewry）**：
    - 全球综合指数及8条主要航线现货价
    - 显示价格和 WoW 变化
    - 点击"更新 WCI"按钮手动刷新
  - **FBX 全球航线指数（Freightos）**：
    - FBX 综合指数及13条航线指数
    - 显示价格和 WoW 变化
    - 点击"更新 FBX"按钮手动刷新

- **AI 分析内容**：
  AI 会基于以下数据进行综合分析：
  1. **派船趋势分析**：派船数量变化趋势和未来预测
  2. **运力变化分析**：运力（TEU）变化趋势和原因分析
  3. **运费走势预测**：基于供需关系推算运费走势
  4. **市场洞察**：市场热点、风险、机会和运营建议
  5. **航线竞争力分析**：不同航线的运力和派船比较
  6. **季节性因素分析**：季节性波动模式和节假日影响
  7. **供需平衡评估**：市场供需平衡状况和变化预测
  8. **市占率与运力对比分析**：对比市占率与各周收货数据，分析供需匹配度
  9. **运费综合考量**：综合用户现行报价、极端报价、SCFI/WCI/FBX 指数、运费走势图等给出涨跌/维持建议

#### AI 分析触发条件

- 必须选择至少一个区域（第一级筛选）
- 必须有可分析的航线数据
- 必须配置有效的 API Key

#### AI 分析数据来源优先级

1. **用户 Excel 数据**：`market` 和 `schedule` 工作表数据
2. **用户自定义输入**："其他影响因素"表格填写的内容
3. **用户上传 PDF**：市场周报（如 Linerlytica Weekly、Alphaliner Newsletter 等）
4. **外部数据抓取**：Ship & Bunker 燃油、CCFI、WCI、FBX 指数
5. **网络公开资料**：当用户数据不足时，AI 会引用公开渠道信息并标注来源

## 详细使用指南

### 第一步：上传数据

1. **上传 Excel 文件**：
   - 点击"选择Excel文件"按钮
   - 选择包含 `market` 和 `schedule` 两个工作表的 Excel 文件
   - 系统会自动解析两个工作表的数据
   - 加载成功后，三个模块的筛选控件将自动显示

2. **上传市场周报（可选）**：
   - 点击"上传市场周报 (PDF)"按钮
   - 可一次选择多份 PDF 文件
   - 支持 Alphaliner Newsletter、Linerlytica Weekly 等格式
   - 上传后会在"市场周报列表"中显示文件名和字数统计
   - PDF 内容会被解析并用于 AI 分析

### 第二步：使用 Module 01 · 运费走势图

1. **选择区域和港口**：
   - 在"选择区域"下拉框中选择一个或多个区域（支持多选）
   - 在"选择港口"下拉框中选择一个或多个港口（支持多选）
   - 港口选项会根据已选区域自动更新

2. **设置时间范围（可选）**：
   - 在图表标题右侧的"开始时间"和"结束时间"中选择日期
   - 系统会自动筛选该时间范围内的数据

3. **查看图表**：
   - 选择港口后，图表会自动更新
   - 信息框显示选中港口数量、时间节点数和每个港口的价格区间
   - 鼠标悬停图表可查看具体数值

### 第三步：使用 Module 02 · 航线运力分析

1. **四级筛选**：
   - 从"区域"开始，逐级选择"子区域"、"国家/地区"、"港口"
   - 每级筛选支持多选，下级选项会根据上级选择自动更新
   - 筛选后表格和图表会自动更新

2. **航线搜索**：
   - 在表头"共舱船公司"列的输入框中输入关键词
   - 支持中文输入，实时过滤
   - 可与四级筛选条件组合使用

3. **切换视图**：
   - 点击"全部"、"运力"、"派船"按钮切换显示模式
   - "全部"模式同时显示运力和派船数
   - "运力"模式仅显示运力（TEU）
   - "派船"模式仅显示派船数量

4. **查看详细信息**：
   - 鼠标悬停表格周别单元格可查看详细船期信息（船名航次、运力、开船日期）
   - 鼠标悬停"共舱船公司"列可查看完整路径信息（对应 schedule 表 Y 列）
   - 筛选后表格底部会显示统计汇总行

5. **查看趋势图**：
   - 选择区域后，页面下方会自动显示运力与派船趋势图
   - 堆积柱形图展示各航线运力分布
   - 折线图展示派船总数变化
   - 图表标签格式：`航线备注 - 共舱船公司`

### 第四步：使用 Module 03 · AI 趋势分析

1. **填写其他影响因素（可选）**：
   - 在"其他影响因素"表格中填写相关信息
   - 收货情况、码头情况、额外运力、市场运费、其他事件
   - 这些信息会被优先用于 AI 分析

2. **更新外部数据（可选）**：
   - 点击"更新燃油价格"按钮获取最新燃油行情
   - 点击"更新 CCFI"按钮获取最新 CCFI 指数
   - 点击"更新 WCI"按钮获取最新 WCI 数据
   - 点击"更新 FBX"按钮获取最新 FBX 指数
   - 数据会缓存6小时，避免频繁请求

3. **配置 AI API**：
   - 在"DeepSeek"、"KIMI (Moonshot)"或"通义千问 (Qwen)"标签页中：
     - 输入 API Key
     - 确认 API URL（通常使用默认值）
     - 选择或输入模型 ID
   - 点击"保存配置"按钮保存设置
   - 配置会自动保存到浏览器本地存储

4. **运行 AI 分析**：
   - 确保已选择至少一个区域（第一级筛选）
   - 点击"使用 DeepSeek 分析"、"使用 KIMI 分析"或"使用通义千问分析"按钮
   - 等待分析完成（通常需要10-30秒）
   - 分析结果会显示在对应标签页中

5. **对比分析结果**：
   - 可以分别运行 DeepSeek、KIMI 和通义千问分析
   - 切换标签页对比三种模型的分析结论
   - 查看观点一致性，交叉验证结论

### 第五步：导出 PDF 报告

1. **准备导出内容**：
   - 完成所有筛选和分析
   - 确保需要导出的内容都已显示在页面上
   - 如需隐藏使用说明，可手动收起

2. **导出 PDF**：
   - 点击"导出整页 PDF"按钮
   - 等待导出完成（通常需要5-15秒）
   - PDF 文件会自动下载，文件名：`运费走势及航线运力分析.pdf`

## AI 分析功能详解

### AI 分析数据来源

AI 分析会综合以下数据源：

1. **Excel 数据**：
   - `schedule` 工作表：上周+本周+未来三周的运力和派船数据
   - `market` 工作表：运费走势图模块选中的港口历史价格数据

2. **用户输入**：
   - "其他影响因素"表格填写的内容
   - 运费走势图模块选中的港口和时间范围

3. **PDF 周报**：
   - 用户上传的市场周报内容
   - 会被解析并优先引用

4. **外部数据**：
   - Ship & Bunker 新加坡燃油价格（VLSFO、MGO、IFO380）
   - 上海航运交易所 CCFI 指数
   - Drewry WCI 主要航线现货价
   - Freightos FBX 全球航线指数

### AI 分析输出结构

AI 分析报告包含以下9个部分：

1. **本周结论**：涨/维/降 + 核心逻辑，明确引用数据源
2. **五大类评分表**：列出维度 / 评分 / 权重 / 加权分 + 总分与判定
3. **需求面分析**：未来两周订舱、装载率预测、山东/沿黄货源走势、季节性、宏观需求
4. **供给面分析**：有效周运力推算、空班/跳港/改道、航班挤压、加开缩减、竞争对手与联盟策略、目的港河港限制
5. **突发事件影响**：红海最新复航预期及其他黑天鹅
6. **内部策略与成本**：HQ 指令、燃油/码头/拖车成本（引用 Ship & Bunker 数据）
7. **SCFI/WCI/FBX 指标解读**：最新指数、WoW/YoY、与现行报价/走势图偏离度及原因
8. **风险提示**
9. **下周展望与可执行建议**：调价、控舱、重柜限制等

## 常见问题

### 数据加载问题

**Q: Excel 文件加载失败怎么办？**

A: 请检查：
- Excel 文件是否包含 `market` 和 `schedule` 两个工作表
- 工作表名称是否正确（不区分大小写，但需包含关键词）
- 数据列是否包含必要的关键词
- 浏览器控制台是否有错误信息

**Q: 为什么筛选控件没有显示？**

A: 筛选控件会在 Excel 文件成功加载 `market` 和 `schedule` 两个工作表后自动显示。如果未显示，请检查：
- Excel 文件是否包含两个工作表
- 工作表名称是否正确
- 浏览器控制台是否有错误信息

### 图表显示问题

**Q: 运费走势图没有显示数据？**

A: 请检查：
- 是否选择了至少一个港口
- 选择的时间范围内是否有数据
- Excel 文件的 `market` 工作表格式是否正确
- 20GP 和 40GP 列名是否正确

**Q: 运力与派船趋势图没有显示？**

A: 请检查：
- 是否选择了至少一个区域（第一级筛选）
- 筛选后是否有可显示的数据
- Excel 文件的 `schedule` 工作表格式是否正确

### AI 分析问题

**Q: AI 分析失败怎么办？**

A: 请检查：
- API Key 是否正确
- 网络连接是否正常
- API 配额是否充足
- API URL 和模型 ID 是否正确
- 是否选择了至少一个区域（第一级筛选）

**Q: AI 分析结果为空或不准确？**

A: 请检查：
- 是否填写了"其他影响因素"表格
- 是否更新了外部数据（燃油、指数等）
- 是否上传了市场周报 PDF
- Excel 数据是否完整

### PDF 导出问题

**Q: PDF 导出失败怎么办？**

A: 请检查：
- 网络连接是否正常（需要加载 html2canvas 和 jsPDF 库）
- 浏览器是否支持相关功能
- 页面内容是否过长（建议先收起使用说明）

**Q: PDF 导出文件太大？**

A: PDF 使用 JPEG 格式压缩，质量设置为 0.65。如果文件仍然较大，可能是页面内容过多。建议：
- 收起使用说明
- 减少筛选的数据量
- 分多次导出不同模块

### 外部数据抓取问题

**Q: 燃油/指数数据抓取失败？**

A: 请检查：
- 网络连接是否正常
- 目标网站是否可访问
- 是否使用了代理或 VPN（可能影响访问）
- 数据会缓存6小时，可稍后重试
