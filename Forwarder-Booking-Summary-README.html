Forwarder · 订舱汇总分析使用说明

基于订舱汇总数据，提供 Sales、Account、船名航次等多个维度的数据分析，支持按部门、周别、GPS、限额等不同维度进行深度分析

工具简介

Forwarder · 订舱汇总分析 是一款专业的订舱数据多维度分析工具，专门为指定货代集团公司设计。支持载入 Excel 文件，通过8个不同维度的分析模式，帮助用户从多个角度深入分析订舱情况，包括 Sales、Account、船名航次等维度的部门分布、周别趋势、GPS分析、限额管理等。工具主要特点：

  • Excel 数据导入：支持载入并解析 Excel 文件，自动识别列结构（需包含 Data 工作表）
  • 8个分析维度：Sales + Dept、Sales + Week、Account + Dept、Account + Week、VSL/VOY + Dept、VSL/VOY + Week、VSL/VOY + GPS、VSL/VOY + Limit
  • 多维度筛选：支持时间范围、Trade、Dept、POD、AREA、SQ/SC、SalesID等多个维度的筛选
  • 详细BL资料：支持二级弹窗查看详细BL资料，包括BLNO、POL/POD、各柜型TEU、GPS、TON等完整信息
  • 数据统计：实时显示总TEU、总TONS、平均GPS、TONS/TEU等关键指标
  • 本地处理：所有数据仅在本地浏览器解析，不会上传至任何服务器

快速开始

  1. 在浏览器中打开工具页面（推荐 Chrome / Edge / Safari）
  2. 点击"载入 Excel"按钮，选择包含 Data 工作表的 Excel 文件
  3. 工具会自动解析数据并显示：
    • 数据状态：显示已加载的记录数
    • 筛选器：自动填充各筛选器的选项
    • 数据摘要：显示总TEU、总TONS、平均GPS、TONS/TEU等关键指标
    • 分析结果：根据选择的维度显示相应的分析结果
  4. 选择分析维度，查看不同角度的数据
  5. 使用筛选器调整分析范围
  6. 点击表格行查看详细资料和BL信息

核心功能

1. 分析维度

工具提供8个不同的分析维度，每个维度从不同角度展示数据：

  • Sales + Dept：按SalesID和部门（AMS/EMS/IAS）分析，显示各柜型分布
  • Sales + Week：按SalesID和周别分析，显示最近4周的订舱趋势
  • Account + Dept：按Account（SQ/SC NAME）和部门分析，显示各柜型分布
  • Account + Week：按Account和周别分析，显示最近4周的订舱趋势
  • VSL/VOY + Dept：按船名航次和部门分析，显示各Trade和POD的分布
  • VSL/VOY + Week：按船名航次和周别分析，显示各Trade和POD的时间分布
  • VSL/VOY + GPS：按船名航次和GPS分析，显示各Trade和POD的GPS情况，支持GPS区间筛选
  • VSL/VOY + Limit：按船名航次和限额分析，显示Allocation、L/F、Limit、Over等限额管理信息

2. 数据筛选

  • 时间范围选择：
    • 支持自定义开始和结束日期
    • 提供快捷按钮：当周+未来3周、最近1个月、最近3个月
  • Trade 筛选（AA列）：支持多选，可搜索筛选
  • Dept 筛选（AB列）：支持多选（AMS/EMS/IAS）
  • POD 筛选（K列）：支持多选，可搜索筛选
  • AREA 筛选（AK列）：支持多选，可搜索筛选
  • SQ/SC 筛选（C列）：支持多选，可搜索筛选
  • SQ/SC NAME 筛选（D列）：已自动硬编码筛选，使用模糊匹配（包含匹配），仅显示指定的5个货代公司，不显示其他SQ/SC name
  • SalesID 筛选（E列）：支持多选，可搜索筛选
  • EXP LINE 筛选（AC列）：支持多选，可搜索筛选

3. 详细资料查看

  • 一级详细资料：
    • 点击主表格的任意行，可查看该行的详细资料
    • 不同维度显示不同的详细内容（如部门分布、周别趋势等）
  • 二级BL详细资料：
    • 在一级详细资料弹窗中，点击任意行可查看该行的详细BL资料
    • 显示内容包括：BLNO、POL/POD、SalesID、SQ/SC、SQ/SC NAME、各柜型TEU（2SD、2RS、4SD、4SH、5SH、4RH、2FO、2FC、4FO、4FC）、TTL TEU、GPS、TON

4. 特殊功能

  • VSL/VOY + GPS 维度：
    • 支持GPS区间筛选（低于平均GPS 90%、80%、70%等）
    • 显示vs Port和vs Type列，对比当前GPS与平均GPS的差异
    • 支持查看详细BL资料，按GPS升序排序
  • VSL/VOY + Limit 维度：
    • 自动从Excel的年份表（如2025、2026等）读取Allocation和Limit数据
    • Allocation列可编辑，修改后会自动保存到浏览器本地存储
    • L/F（Load Factor）自动计算：TEU / Allocation × 100%
    • Over列显示超舱情况，超过15的会以橙色背景标识
    • 支持目的港排除筛选

数据列说明

工具使用的Excel数据列映射如下：

  • A列：BLNO（提单号）
  • C列：SQ/SC
  • D列：SQ/SC NAME
  • E列：SalesID
  • G列：VESSEL（船名）
  • H列：VOYAGE（航次）
  • I列：POL（起运港）
  • K列：POD（目的港）
  • L列：DLY
  • N~W列：各柜型TEU（2SD、2RS、4SD、4SH、5SH、4RH、2FO、2FC、4FO、4FC）
  • X列：TTL TEU（总TEU）
  • Y列：TONS（吨数，需/1000）
  • Z列：GPS
  • AA列：Trade
  • AB列：Dept（AMS/EMS/IAS）
  • AC列：EXP LINE
  • AD列：SAIL DATE（开船日期）
  • AE列：WEEK（周别）
  • AK列：AREA

使用技巧

  • 快速切换维度：点击顶部的维度选择按钮，快速切换不同的分析维度
  • 组合筛选：可以同时使用多个筛选器，实现精确的数据筛选
  • 查看详细数据：点击表格行可以逐级查看详细资料，从汇总到明细
  • GPS分析：在VSL/VOY + GPS维度中，使用GPS区间筛选快速找出异常数据
  • 限额管理：在VSL/VOY + Limit维度中，可以编辑Allocation值，系统会自动保存并重新计算L/F
  • 数据持久化：修改的Allocation值会保存到浏览器本地存储，刷新页面后仍然保留

常见问题

Q: 为什么数据没有显示？

A: 请确保已上传 Excel 文件并包含 Data 工作表。如果数据已加载但没有显示，请检查筛选条件是否过于严格导致没有匹配的数据。

Q: Allocation值如何修改？

A: 在VSL/VOY + Limit维度中，直接点击Allocation列的数值，可以编辑。修改后会自动保存到浏览器本地存储，并重新计算L/F。

Q: 如何查看详细BL资料？

A: 点击主表格的任意行，会弹出详细资料弹窗。在详细资料弹窗中，再次点击任意行，会弹出该行的详细BL资料。

Q: GPS区间筛选如何使用？

A: 在VSL/VOY + GPS维度的详细BL资料弹窗中，可以选择"低于平均GPS 90%"、"80%"、"70%"等多个选项，系统会筛选出符合条件的BL记录。

Q: 数据刷新后Allocation修改会丢失吗？

A: 不会。Allocation的修改会保存到浏览器本地存储，即使刷新页面或重新上传Excel文件，修改的值仍然会保留。

Q: 如何排除特定目的港？

A: 在VSL/VOY + Limit维度的详细资料弹窗中，可以使用目的港排除选项，勾选要排除的POD，表格会自动更新。
