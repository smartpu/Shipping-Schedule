工具简介

Market Information · 企业宣传资料 是一款 PDF 文档查看工具，用于展示和查看企业宣传介绍、OA Day 活动资料、市场周报等相关文档。工具主要特点：

  • 自动识别文件：从 pdf/index.json 自动读取文件列表，无需手动维护
  • 智能分类：根据文件名自动分类为企业介绍、OA Day、市场周报等类别
  • 全屏查看：PDF 打开时自动进入全屏模式，提供沉浸式阅读体验
  • 智能显示模式：
    • 航运周报（竖版PDF）：左右两页并排显示，模拟真实阅读体验
    • 企业介绍和OA资料（横版PDF）：全屏单页显示，适合演示和展示
  • 键盘操作：支持上下键或 Page Up/Down 翻页，ESC 键退出
  • 文件管理：只需在 pdf/index.json 中添加文件名，系统自动识别和分类

快速开始

  1. 在浏览器中打开工具页面（推荐 Chrome / Edge / Safari）
  2. 工具会自动从 pdf/index.json 加载 PDF 文件列表
  3. 文件按类别分组显示：
    • 企业介绍：企业宣传介绍文档（中英文版本）
    • OA Day 活动资料：历年 OA Day 活动相关文档
    • 市场周报：Linerlytica、Alphaliner 等市场周报
    • 其他资料：其他 PDF 文档
  4. 点击任意 PDF 卡片即可全屏查看
  5. 使用键盘上下键或 Page Up/Down 切换不同的 PDF 文件
  6. 按 ESC 键退出全屏

核心功能

1. 文件自动识别

工具会自动从 pdf/index.json 读取文件列表。文件按类别组织，系统会自动识别和分类显示。

文件类别：

  • 市场周报：市场分析报告文件列表
  • 企业介绍：企业宣传介绍文档列表
  • OA Day 活动资料：OA Day 活动相关文档列表

2. 智能分类

系统会根据文件名自动分类：

  • 企业介绍：文件名包含 "introduction" 的文件
    • 自动识别中英文版本（CN/Chinese/中文 或 EN/English/英文）
  • OA Day：文件名包含 "oa-day"、"oa_day"、"oaday" 的文件
    • 自动提取年份信息
    • 自动识别多部分文档（part1、part2 等）
    • 按年份倒序排列
  • 市场周报：文件名包含 "linerlytica"、"alphaliner"、"weekly"、"report"、"周报" 的文件
  • 其他资料：其他未分类的 PDF 文件

3. 显示模式

  • 竖版PDF（航运周报）：
    • 自动识别 Linerlytica、Alphaliner 等航运周报
    • 采用左右两页并排显示模式
    • 深色背景，白色 PDF 区域
    • 适合阅读竖版周报文档
  • 横版PDF（企业介绍、OA资料）：
    • 自动识别企业介绍、OA Day 等横版文档
    • 采用全屏单页显示模式
    • 适合演示和展示

4. 键盘操作

  • ↑ 键 / Page Up：切换到上一个 PDF 文件
  • ↓ 键 / Page Down：切换到下一个 PDF 文件
  • ESC 键：退出全屏或关闭查看器
  • F 键：切换全屏模式（如果浏览器支持）

5. 文件管理

添加新文件：

  1. 将 PDF 文件放入 pdf/ 目录
  2. 在 pdf/index.json 的对应数组中添加文件名
  3. 刷新页面，新文件会自动显示并分类

操作步骤：

  • 添加新的市场周报：在对应类别数组中添加文件名
  • 添加新的企业介绍：在对应类别数组中添加文件名
  • 添加新的 OA Day 资料：在对应类别数组中添加文件名

注意事项

  • 文件格式：仅支持 PDF 格式文件
  • 文件路径：所有 PDF 文件必须放在 pdf/ 目录下
  • 索引文件：必须维护 pdf/index.json 文件，工具才能自动识别文件
  • 浏览器兼容性：推荐使用 Chrome、Edge 或 Safari 浏览器
  • 全屏功能：需要浏览器支持全屏 API，某些浏览器可能需要用户手动授权
  • 左右分页显示：竖版PDF的左右分页功能取决于浏览器对PDF查看器的支持，某些浏览器可能显示为单页

常见问题

Q: 为什么我的PDF文件没有显示？

A: 请检查：

  • 文件是否已添加到 pdf/index.json 的对应数组中
  • 文件名是否正确（区分大小写）
  • 文件是否存在于 pdf/ 目录下
  • 刷新页面后是否显示

Q: 如何更新市场周报？

A: 只需：

  1. 删除旧的周报文件（可选）
  2. 将新的周报文件放入 pdf/ 目录
  3. 在 pdf/index.json 的 marketReports 数组中更新文件名
  4. 刷新页面即可看到新文件

Q: 左右两页显示不工作怎么办？

A: 左右两页显示功能依赖于浏览器对PDF查看器的支持。如果无法正常显示：

  • 尝试使用 Chrome 或 Edge 浏览器
  • 检查浏览器是否支持PDF内嵌显示
  • 某些浏览器可能会自动下载PDF而不是显示，这是正常行为

Q: 可以同时查看多个PDF吗？

A: 当前版本一次只能查看一个PDF。使用键盘上下键可以快速切换不同的PDF文件。
