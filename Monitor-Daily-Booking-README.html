工具简介

Monitor · 日订舱监控 是一款专业的订舱数据分析工具。支持载入 Excel 文件，通过多维度筛选（时间范围、BKG Party、POD、FTB、Trade）快速分析订舱情况，提供 Trade 维度和客户维度两种分析模式，帮助用户快速掌握订舱趋势并做出决策。工具主要特点：

  • Excel 数据导入：支持载入并解析 Excel 文件，自动识别列结构（需包含 Daily Booking 工作表）
  • 多维度筛选：支持时间范围、BKG Party、POD、FTB、Trade 五个维度的筛选，筛选条件联动
  • 双模式分析：支持 Trade 维度和客户维度两种分析模式，满足不同分析需求
  • 数据可视化：提供饼图、堆叠柱状图等多种图表类型，直观展示数据分布和趋势
  • 高级功能：数据对比、排名视图、筛选预设、数据快照等实用功能
  • 数据导出：支持导出 Excel 和 PDF 格式，包含完整图表和表格

快速开始

  1. 在浏览器中打开工具页面（推荐 Chrome / Edge / Safari）
  2. 点击"载入 Excel"按钮，选择包含 Daily Booking 工作表的 Excel 文件
  3. 工具会自动解析数据并显示：
    • 数据状态：显示已加载的记录数
    • 筛选器：自动填充各筛选器的选项
    • 数据摘要：显示总BL、总TEU、各状态占比等关键指标
    • 分析结果：根据当前模式显示 Trade 维度或客户维度的分析结果
  4. 使用筛选器调整分析范围，查看不同条件下的数据
  5. 切换分析模式，从不同维度查看数据
  6. 点击"导出Excel"或"导出PDF"按钮，导出分析结果

核心功能

1. 数据筛选

  • 时间范围选择：
    • 支持自定义开始和结束日期
    • 提供快捷按钮：最近1周、最近1个月、最近3个月
    • 默认显示最近1个月的数据
  • BKG Party 筛选（F列）：支持多选，可搜索筛选
  • POD 筛选（K列）：支持多选，可搜索筛选
  • FTB 筛选（X列）：支持多选，可搜索筛选
  • Trade 筛选（Z列）：支持多选，可搜索筛选
  • 筛选预设：
    • 点击"保存筛选预设"保存当前筛选条件为模板
    • 点击"管理预设"查看、应用或删除已保存的预设
    • 预设会自动保存到浏览器本地存储

2. 分析模式

  • 模式1：Trade维度：
    • 按 Trade 和时间分组展示订舱情况
    • 显示每个 Trade 在不同时间段的订舱情况
    • 默认按时间升序排序
    • 支持按总量、A/C/F/L、N/P、X 等指标排序
  • 模式2：客户维度：
    • 按 BKG Party 分组展示订舱情况
    • 显示每个客户的订舱统计
    • 默认按总量降序排序
    • 支持按总量、A/C/F/L、N/P、X 等指标排序

3. 数据统计指标

每个分组都会显示以下统计指标：

  • 总量：BL（记录数）和 TEU（标准箱数）
  • A/C/F/L：状态为 A、C、F、L 的订舱统计（BL 和 TEU）及占比
  • N/P：状态为 N、P 的订舱统计（BL 和 TEU）及占比
  • X：状态为 X 的订舱统计（BL 和 TEU）及占比

4. 数据可视化

  • 饼图：展示 A/C/F/L、N/P、X 三种状态的占比分布
  • 堆叠柱状图（按时间）：展示每日的 TEU 分布，按状态堆叠显示
  • 堆叠柱状图（按分组）：
    • 模式1：按 Trade 展示各 Trade 的 TEU 分布
    • 模式2：按 BKG Party 展示各客户的 TEU 分布
  • 图表交互：
    • 点击图例可显示/隐藏对应的数据系列
    • 悬停查看详细数值

5. 高级功能

  • 数据对比：
    • 点击"数据对比"按钮打开对比面板
    • 选择对比的开始和结束日期
    • 点击"应用对比"查看当前期间与对比期间的详细对比
    • 显示变化量和变化百分比，用颜色标识增长/下降
  • 排名视图：
    • 点击"排名视图"按钮打开排名面板
    • 选择显示数量（Top 10/20/50 或全部）
    • 选择排序依据（总BL、总TEU、A/C/F/L BL/TEU）
    • 点击"应用排名"应用筛选，表格中显示排名徽章
  • 数据钻取：
    • 点击表格行查看该分组的详细记录
    • 弹窗显示所有原始数据记录
  • 数据刷新：
    • 点击"刷新数据"按钮重新读取当前 Excel 文件
    • 自动保持当前筛选条件
  • 数据快照：
    • 点击"保存快照"保存当前分析结果为快照
    • 点击"快照对比"查看所有快照
    • 支持与上一个快照对比，查看数据变化
    • 最多保留 20 个快照

6. 数据导出

  • 导出 Excel：
    • 导出当前模式的分析结果
    • 包含完整的表格数据
    • 自动生成文件名（包含日期和模式信息）
  • 导出 PDF：
    • 导出当前模式的分析结果
    • 包含所有图表（饼图、堆叠柱状图）和表格
    • 自动分页，支持多页内容
    • 自动生成文件名（包含日期和模式信息）

数据格式要求

  • Excel 文件需包含名为 "Daily Booking" 的工作表
  • A 列：日期（支持 Excel 日期序列号和标准日期格式）
  • C 列：订舱状态（A/C/F/L/N/P/X）
  • F 列：BKG Party（订舱方）
  • K 列：POD（目的港）
  • X 列：FTB（首航港）
  • Y 列：TEU（标准箱数，数值）
  • Z 列：Trade（贸易航线）
  • 数据按日期排列，支持多行数据

使用技巧

  • 筛选条件记忆：工具会自动保存筛选条件，下次打开页面时会自动恢复（7天内有效）
  • 筛选预设：将常用的筛选组合保存为预设，一键应用，提高工作效率
  • 数据快照：定期保存快照，便于对比不同时期的数据变化
  • 排名视图：使用排名功能快速找出 Top 客户或 Trade
  • 数据对比：对比不同时间段的数据，分析趋势变化
  • 数据钻取：点击表格行查看详细记录，深入了解数据构成
  • 图表交互：点击图例显示/隐藏数据系列，聚焦关注的数据

注意事项

  • 所有数据仅在本地浏览器解析，不会上传至任何服务器
  • 解析结果仅供内部研判使用，不构成对外报价或投资建议
  • 建议使用最新版本的 Chrome、Edge 或 Safari 浏览器以获得最佳体验
  • 大数据量（超过 10,000 条记录）时，解析可能需要一些时间，请耐心等待
  • 筛选预设和快照数据保存在浏览器本地存储中，清除浏览器数据会丢失
  • PDF 导出包含图表，文件大小可能较大，导出时请耐心等待

常见问题

Q: 为什么图表没有显示？

A: 请确保已上传 Excel 文件并包含有效数据。如果数据已加载但图表未显示，请检查筛选条件是否过于严格导致没有匹配的数据。

Q: 如何清除所有筛选条件？

A: 当筛选后没有数据时，会显示"清除所有筛选条件"按钮。也可以手动清除各筛选器的选择，或使用筛选器下方的"清除选择"按钮。

Q: 筛选预设保存在哪里？

A: 筛选预设保存在浏览器的本地存储（localStorage）中，只在本浏览器中有效。清除浏览器数据会丢失所有预设。

Q: PDF 导出时图表不完整怎么办？

A: 请确保在导出前图表已完全渲染。如果图表不完整，可以等待几秒后重试，或刷新页面重新加载数据。

Q: 数据刷新后筛选条件会丢失吗？

A: 不会。数据刷新功能会自动保持当前筛选条件，刷新后会自动应用之前的筛选设置。
