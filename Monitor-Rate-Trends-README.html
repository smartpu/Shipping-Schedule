Monitor · 运价趋势面板使用说明

工具简介

这个工具用于快速查看运价趋势，方便直接在浏览器中上传 Excel 并观察 2SD / 4SH 运价走势，无需安装任何额外环境。

功能特点

  • 自动读取数据：支持上传运价 Excel 文件，自动识别 D 列日期、F 列港口、G 列运费类型（2SD/4SH）和 H 列运费
  • 多维度筛选：支持按时间范围和港口筛选数据
  • 多种图表展示：筛选区间趋势曲线、全历史周度走势、年度对比同期观察
  • 周度涨跌统计：自动计算并显示周度涨跌情况

使用步骤

  1. 在浏览器中打开 Monitor-Rate-Trends.html（推荐 Chrome / Edge / Safari）
  2. 点击"上传 Excel"选择运价 Excel 文件（需包含 D 列日期、F 列港口、G 列运费类型、H 列运费）
  3. 通过时间、港口筛选内容，浏览筛选区间、全历史、年度对比等图表，并查看周度涨跌统计

数据格式要求

  • D 列：日期（支持 Excel 日期序列号和标准日期格式）
  • F 列：港口名称
  • G 列：运费类型（2SD 或 4SH）
  • H 列：运费数值

注意事项

  • 所有数据仅在本地浏览器解析，不会上传至任何服务器
  • 解析结果仅供内部研判，不构成对外报价或投资建议
  • 建议使用最新版本的 Chrome、Edge 或 Safari 浏览器以获得最佳体验
