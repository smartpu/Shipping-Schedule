工具简介

这个工具用于快速查看 SCFI（上海出口集装箱运价指数）的历史趋势，方便直接在浏览器中载入 Excel 文件并观察各指数走势，无需安装任何额外环境。

功能特点

  • 自动读取数据：支持载入 SCFI Excel 文件，自动识别 B 列到 Q 列的综合指数及分 trade 指数
  • 历史趋势图：展示完整的 SCFI 历史趋势，支持自定义时间范围筛选
  • 年度对比图：展示最近 3 年的年度对比趋势，便于同期观察和对比分析
  • 指数选择：
    • 历史趋势图：支持单选模式，一次只显示一个指数，便于清晰观察
    • 年度对比图：支持多选模式，可同时对比多个指数在不同年份的走势
    • 默认显示"SCFI Comprehensive Container Freight Rate Index"（综合指数）
  • 自定义筛选：支持手动调整开始和结束日期，查看不同时间段的数据
  • 优化图表尺寸：图表高度经过优化，在保证清晰度的同时提高页面利用率

使用步骤

  1. 在浏览器中打开工具页面（推荐 Chrome / Edge / Safari）
  2. 工具会自动从配置文件读取并加载对应的 Excel 测试数据文件（如果配置存在）
  3. 如果自动加载失败或需要手动选择：
    • 点击"载入 Excel"按钮，选择 SCFI Excel 文件
  4. 工具会自动加载数据并显示：
    • SCFI 历史趋势图：显示完整的历史数据，默认显示综合指数
    • 年度对比图：显示最近 3 年的年度对比，便于同期观察
    • 指数对比分析：选择对比日期，查看各指数在不同时间点的对比数据
  5. 在历史趋势图中，点击图例可切换显示不同的指数（单选模式）
  6. 在年度对比图中，点击图例可同时显示多个指数进行对比（多选模式）
  7. 如需查看其他时间段，可通过日期选择器调整开始和结束日期
  8. 年度对比图的右上角显示年份颜色说明，便于识别不同年份
  9. 点击右上角"导出PDF"按钮，可将所有图表导出为PDF文件

数据格式要求

  • Excel 文件第一行为列名
  • A 列：日期（支持 Excel 日期序列号和标准日期格式）
  • B 列到 Q 列：各指数数值（综合指数及分 trade 指数）
  • 数据按日期升序排列

注意事项

  • 所有数据仅在本地浏览器解析，不会上传至任何服务器
  • 解析结果仅供内部研判，不构成对外报价或投资建议
  • 建议使用最新版本的 Chrome、Edge 或 Safari 浏览器以获得最佳体验
