## SCFI 历史趋势说明

这个工具用于快速查看 SCFI（上海出口集装箱运价指数）的历史趋势，方便直接在浏览器中上传 Excel 文件并观察各指数走势，无需安装任何额外环境。

### 功能特点

- **自动读取数据**：支持上传 SCFI Excel 文件，自动识别 B 列到 Q 列的综合指数及分 trade 指数
- **默认时间范围**：自动显示最近 3 年的数据趋势
- **自定义筛选**：支持手动调整开始和结束日期，查看不同时间段的数据
- **大尺寸图表**：图表高度为两倍，便于查看详细趋势和对比分析
- **多指数展示**：同时展示多个指数的趋势线，支持自定义图例

### 使用步骤

1. 在浏览器中打开 `Monitor-SCFI-Trends.html`（推荐 Chrome / Edge / Safari）
2. 点击"选择 SCFI.xlsx 文件"按钮，选择 SCFI Excel 文件
3. 工具会自动加载数据并显示最近 3 年的趋势图
4. 如需查看其他时间段，可通过日期选择器调整开始和结束日期
5. 图表下方显示所有指数的图例，便于识别不同指数

### 数据格式要求

- Excel 文件第一行为列名
- A 列为日期（支持 Excel 日期序列号和标准日期格式）
- B 列到 Q 列为各指数数值（综合指数及分 trade 指数）
- 数据按日期升序排列

### 注意事项

- 所有数据仅在本地浏览器解析，不会上传至任何服务器
- 解析结果仅供内部研判，不构成对外报价或投资建议
- 建议使用最新版本的 Chrome、Edge 或 Safari 浏览器以获得最佳体验
