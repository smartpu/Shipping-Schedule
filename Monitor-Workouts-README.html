Monitor · 健身记录面板使用说明

工具简介

健身记录可视化工具，用于分析苹果健身记录数据，追踪运动距离和时间趋势。支持载入 CSV 格式的健身数据，通过图表展示运动趋势，帮助您了解自己的运动习惯和进步情况。

功能特点

  • CSV 数据导入：支持载入 CSV 格式的健身记录数据
  • 趋势可视化：使用图表展示运动距离和时间趋势
  • 多视图切换：支持按不同时间维度查看数据（日/周/月/年）
  • 数据聚合：自动聚合数据，便于查看不同时间段的统计信息
  • PDF 导出：支持将图表导出为 PDF 文件
  • 本地处理：所有数据仅在本地浏览器解析，不会上传至任何服务器

使用步骤

  1. 在浏览器中打开 Monitor-Workouts.html（推荐 Chrome / Edge / Safari）
  2. 点击"载入 CSV"按钮，选择您的健身记录 CSV 文件
  3. 工具会自动解析数据并显示：
    • 数据状态：显示已加载的数据条数
    • 健身记录趋势图表：展示运动距离和时间的变化趋势
  4. 使用视图切换按钮（日/周/月/年）查看不同时间维度的数据聚合结果
  5. 如需导出图表，点击"导出 PDF"按钮即可保存为 PDF 文件

数据格式要求

  • 健身记录文件必须是 CSV 格式
  • CSV 文件应包含以下列（具体列名可能因数据源而异）：
    • 日期：运动记录的日期
    • 距离：运动距离（单位：公里或米）
    • 时间：运动时长（单位：分钟或小时）
    • 其他相关字段（如运动类型、卡路里等，根据实际数据而定）
  • 数据按日期升序排列效果最佳
  • 示例格式： 
    日期,距离(km),时间(分钟),类型
2025/01/01,5.2,30,跑步
2025/01/02,3.5,25,步行
2025/01/03,10.0,60,骑行
...

视图说明

  • 日视图：按天显示每日的运动数据
  • 周视图：按周聚合数据，显示每周的运动统计
  • 月视图：按月聚合数据，显示每月的运动统计
  • 年视图：按年聚合数据，显示每年的运动统计

注意事项

  • 所有数据仅在本地浏览器解析，不会上传至任何服务器
  • 建议使用最新版本的 Chrome、Edge 或 Safari 浏览器以获得最佳体验
  • 如果 CSV 文件较大，加载和解析可能需要几秒钟时间
  • 图表需要 Chart.js 库支持，如果图表无法显示，请检查网络连接或刷新页面
  • PDF 导出功能需要 html2canvas 和 jsPDF 库支持，首次使用时会自动加载
  • 确保 CSV 文件格式正确，日期和数值字段格式规范，否则可能导致解析失败
